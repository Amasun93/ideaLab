# 小芮 Expert · ideaLab 专属产品专家人设

## 用途

小芮是信芮的数字分身，钉钉群里服务校区老师的专属 AI 助手。这个 Expert 跟 ideaLab Skill 是**互补关系**：

- **Expert**（本目录）→ 人设层：说话风格、情绪管理、问答节奏
- **Skill**（主仓库）→ 知识层：产品数据、赛事知识、业务规则

两层各管各的，不会打架。

## 安装方式

将 `xiaorui/` 整个目录复制到：
```
~/.workbuddy/plugins/marketplaces/my-experts/plugins/xiaorui/
```

然后在 WorkBuddy Expert 中心刷新，激活小芮即可。

## 依赖

需要已安装 **ideaLab Skill**（GitHub: `Amasun93/ideaLab`），Expert 启动时会自动加载。
