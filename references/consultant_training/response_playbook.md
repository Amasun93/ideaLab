# 回复思路与话术思路 Playbook

更新时间：2026-07-02

本文件用于沉淀可复用的回复方法、话术结构和顾问表达原则。它不是产品事实库，也不是逐字稿库。

当同事说“这个回复思路记一下”“以后这种问题要这样问”“这类家长要先承接再推荐”时，优先判断是否应该进入本文件。

## 使用场景

读取本文件：

- 用户问“这类家长怎么回复”“话术思路是什么”“怎么开口更好”。
- 用户要求示范顾问回答。
- 用户要求复盘顾问回答。
- 用户说“这个回复逻辑记一下”“以后类似问题按这个思路”。
- Agent 需要把产品/赛事/校情事实转成家长能听懂的话。

不要用本文件替代事实核查。价格、课时、退费、赛事节点、学校属性等事实必须回到对应资料卡。

## 总原则

好的回答不是背话术，而是完成四件事：

1. 听懂家长真实目标。
2. 判断孩子当前阶段。
3. 把产品或赛事放进一条可理解的路径。
4. 守住不保奖、不承诺升学、不编数据的边界。

默认回答顺序：

```text
承接情绪/目标 -> 给初步判断 -> 补关键追问 -> 给阶段路径 -> 匹配产品 -> 讲清边界 -> 递下一步话头
```

## 家长端表达结构

### 1. 承接

先接住家长目标或担心，不要直接进入产品报价。

可用方向：

- “您这个目标其实要分阶段看。”
- “如果是冲三公/强校，就不能只看现在上一门课，要看孩子能不能持续产出项目证据。”
- “这个问题问得很关键，因为启航、专项营和竞赛营解决的是不同问题。”

避免：

- 一上来报价格。
- 一上来推最高价产品。
- 用内部词解释内部词。

### 2. 判断目标

先判断家长属于哪类需求：

- 兴趣启蒙：希望孩子喜欢动手、敢表达。
- 能力培养：希望长期做原创项目。
- 结果导向：希望有赛事申报和成果。
- 升学素材：希望围绕三公、强校、自招、综评建立项目证据。
- 价格敏感：先确认预算和目标优先级。

如果目标不清楚，先追问，不急着推荐。

### 3. 补关键追问

优先追问：

- 孩子年级、城市/区和目标学校。
- 是否喜欢动手、搭建、观察生活问题、讲作品。
- 专注时长和遇到失败是否愿意再试。
- 家庭目标是兴趣、三公/强校、赛事、综评还是长期能力。
- 可投入周期和预算区间。

不要问显而易见的问题。例如中班家长咨询时，不要问“几年级进入小学”。

### 4. 给路径

路径要按阶段讲，不要只给产品名。

示例结构：

```text
现在这个阶段，先看兴趣和动手基础；
到三年级以后，重点开始做原创项目；
四五年级如果目标三公/强校，再看雏鹰杯专项营或寒(暑)假竞赛营；
到初中阶段，再根据孩子项目基础进入领航或更高阶赛事服务。
```

### 5. 匹配产品

推荐时必须说明：

- 为什么推荐这个。
- 为什么暂时不推荐另一个。
- 当前推荐依赖哪些前提。

不要只说“适合报启航”。

更好的表达：

```text
如果孩子现在更需要建立原创项目意识，启航会更合适；
如果已经有项目基础，并且四五年级明确要冲三公/强校材料，再考虑雏鹰杯专项营或竞赛营。
```

### 6. 补边界

涉及比赛、升学、退费和合同，必须补边界：

- 赛事结果不承诺。
- 升学结果不承诺。
- 价格、排班、合同、退费以校区签约和当期 SOP 为准。
- 退费承诺不是获奖承诺。

## 规划类问题

当家长问“孩子怎么规划”“有什么课程推荐”“现在中班/小学/初中怎么走”：

1. 信息不足先追问。
2. 默认按上海家长理解，除非用户说明其他城市。
3. 低龄不直接推赛事服务。
4. 四五年级且目标三公/强校/赛事成果时，考虑雏鹰杯专项营或寒(暑)假竞赛营，但前提是项目基础和表达能力达标。
5. 价格表达用“目前售价”，不要对家长说“当前口径”。

## 价格类问题

家长问“怎么收费”“有没有课程推荐，多少钱”时：

1. 先确认目标和孩子阶段。
2. 再给产品选择。
3. 最后说价格。

推荐表达：

```text
如果目标是先做能力和原创项目，主要看启航/领航；
如果目标已经明确到赛事申报和材料服务，就看专项营或寒(暑)假竞赛营。
目前售价我可以先给您一个参考，具体排班、合同和退费以校区签约为准。
```

避免：

- “当前口径是……”
- “这个包保证更有用。”
- “报这个就能拿奖/对升学一定有帮助。”

## 赛事服务类问题

家长问“能不能冲奖”“自己投行不行”“竞赛营和专项营怎么选”时：

- 先区分能力课和赛事服务。
- 说明赛事服务的价值是选题、材料、申报、查新、答辩和节奏管理。
- 结果仍取决于学生投入、家长配合、学校配合、赛事规则和评审偏好。
- 预算够且多赛事目标明确，看寒(暑)假竞赛营。
- 预算有限或单一赛事目标明确，看专项营。

## 校情与赛情问题

校情回答：

- 先判断学校/城市是否重视科创。
- 再拆学校属性、课程/班型、科创信号、升学/材料信号。
- 最后给低龄、四五年级、初中/高中分阶段规划。

赛情回答：

- 围绕区、年份、赛事、申报窗口、推荐机制、材料限制、评审节点。
- 不暴露学生名单、证书、学籍号、老师信息或原始通知明细。
- 没有当年通知时，说明需要核对最新官方/区级通知。

## 情绪承接

家长焦虑时，先降噪，再规划。

可用方向：

- “您现在担心的是时间窗口会不会错过，这个确实要提前看。”
- “不用一上来就冲最高难度，我们先看孩子基础和家庭目标。”
- “如果孩子还小，真正要守住的是兴趣、动手和表达，不是提前把赛事压力压上去。”

避免：

- 否定家长焦虑。
- 夸大紧迫感。
- 用“错过就没机会”制造压力。

## 内部与家长端转换

内部可以说：

- “这个家庭结果导向强。”
- “孩子项目基础不够，不适合直接推竞赛营。”
- “这所学校是强校信号，适合提前做科创证据链。”

对家长要转换为：

- “我们先看孩子目前有没有能持续打磨的项目基础。”
- “这类学校通常会更看重孩子有没有长期、真实、能讲清楚的项目经历。”
- “如果目标比较明确，四五年级可以开始看赛事型路径，但前提是孩子能做、能改、能讲。”

## 同事说“记一下”时怎么入库

| 同事补充内容 | 入库位置 |
|---|---|
| 回复顺序、开口方式、追问方式 | 本文件 |
| 某类家长常问问题和安全回答 | `parent_faq_bank.json` |
| 顾问考核扣分点 | `scoring_rubric.json` |
| 具体产品事实 | `references/product_knowledge/` |
| 具体赛事事实或赛情 | `references/event_knowledge/` |
| 具体校情 | `references/school_knowledge/` |
| 新问题类型的读取路径 | `SKILL.md` |
| 新入库流程或字段规范 | `WIKI.md` / `_governance/` |

## 禁用表达

- 保奖、保证获奖、保证进三公、保证综评有用。
- “官方获奖率就是……”，除非有明确官方来源和统计口径。
- “这个学校一定看科创。”
- “内部资料显示……”，面向家长不这样说。
- “当前口径是……”，面向家长改成“目前售价/目前资料看/目前校区执行以签约为准”。

## 快速自检

回答发出前检查：

- 有没有先判断家长目标？
- 有没有补必要追问？
- 有没有把产品放进路径，而不是只报产品名？
- 有没有说明推荐前提？
- 有没有守住价格、合同、赛事、升学边界？
- 有没有把内部词换成家长能听懂的话？
