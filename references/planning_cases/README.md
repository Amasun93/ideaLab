# 规划案例库

本目录沉淀真实的学员规划案例，用于后续相似场景快速参考。每个案例按标准模板编写。

## 标准模板

```markdown
# [案例标题]

- **日期**：记录时间
- **学生画像**：年级 / 学校类型 / 地区 / 核心优势或短板
- **家庭诉求**：升学方向 / 预算 / 时间 / 家长类型
- **推荐方案**：推荐的产品及理由
- **备选方案**：如果推荐方案不成立，退一步的选择
- **关键判断点**：为什么选A不选B的核心逻辑
- **边界提醒**：有什么坑要注意，有什么话术不要说
```
