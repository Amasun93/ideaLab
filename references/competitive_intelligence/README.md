# 竞品情报库

本目录存放竞品资料卡，每张卡片覆盖一个竞品机构。

## 标准字段

| 字段 | 说明 |
|------|------|
| 机构名称 | 竞品全称 |
| 定位 | 一句话核心业务 |
| 课程/产品 | 主要产品线 |
| 价格 | 收集到的价格信息（注明时间） |
| 目标群体 | 年龄/学段 |
| 核心卖点 | 营销话术中的关键优势 |
| 与 ideaLab 直接竞争的产品 | 映射到我们的哪个产品 |
| 竞品劣势 | 他们的短板、风险点、家长可能担心的问题 |
| 我方优势 | 跟竞品比我们好在哪 |
| 注意事项 | 家长可能会拿来比较的点，怎么说 |
| 来源 | 信息来自哪里（官网/同事反馈/家长提及/内部对比表） |

## ideaLab 五大销售话术支柱（跨竞品通用）

当家长拿任何竞品来比较时，从这五个角度回应（顺序可灵活）：

1. **科创项目经验、赛事经验丰富**
   - 业内唯一一家科研素养经验能够覆盖全国 15 个省市的机构
   - 每年项目总数上千个，奖项数量也上千
   - 区别于"赛事经验仅限本市"的机构

2. **全职师资团队，教学经验丰富**
   - 极具特色的"交叉学科"教练团队，且均为全职
   - 核心教研团队具备 10 年相关研发经验，均毕业于国内外名校
   - 区别于"其他机构全职兼职，鱼龙混杂，多为大学生兼职带项目"

3. **坚持学生主导的项目原则**
   - 教会孩子从个人兴趣出发，学会自主推进项目
   - 区别于市面上的"代打"（把高校/国外博士论文拆出来直接找比赛）

4. **环境设备**
   - 高端的创客实验室环境，助力孩子学习以及项目的制作
   - 区别于市面上的"其他机构、无完善的实验室以及对应的设备"

5. **一站式的辅导规划**
   - 小初高都有对应的培养课程
   - 市面上的机构基本只有小学，初中，大部分家长都是遵循机构的培养体系，会出现高年龄断没有规划
   - ideaLab 是有完整学段的

## 竞品分类速记

| 类别 | 竞品 | 共性 |
|------|------|------|
| **高价理论派** | 智勇 | 偏理论课程、师资兼职、运营不稳 |
| **学科转型派** | 昂立科创学院 | 借鉴 ideaLab 学料、师资外包、奖项不承诺结果 |
| **低价入门派** | 倍塔狗 | 班课+1v1、便宜、无申报服务、家长 DIY |
| **团队工程派** | Wowkids | 双语、3 人团队、高端无奖项承诺、用我方旧课件 |

## 竞品运营/合规风险（敏感话题）

- **智勇**："近半年发不出工资"属于公开可表达信息，销售时可以如实提及，**不构成商业诋毁**（陈述事实 + 来源 = 内部对比表）
- **昂立科创学院**："抄袭/借鉴"——对外避免直接用"抄袭"二字，改用"借鉴/学习"，避免法律风险
- **Wowkids 课件历史**：不要主动说"他们用了我们 3-4 年前的课件"
- 所有"竞品劣势"在表达时保持"客观事实 + 数据来源"风格，不做情绪化对比

## 使用方式

- 被问到"XX机构跟你们比怎么样"时读对应卡片
- 卡片不全的标记 `TODO`，由大卫老师或同事补全
- 信息有新的变化时更新对应字段
- 销售话术沉淀后补充到 `consultant_training/parent_faq_bank.json`（需走低风险更新流程）
