# 昂立科创学院

- **机构名称**：昂立科创学院
- **定位**：学科扩科（传统培训机构转型做科创）
- **课程/产品**：学料扩科类课程
- **价格**：收费 55,800；5 个赛事中任选 3 个赛事申报（青创、赛复、明科、小研究员、雏鹰杯）；0 获奖退费 50%
- **目标群体**：中小学生科创/赛事家庭
- **核心卖点**：5 选 3 多赛事申报、零获奖退一半
- **与 ideaLab 直接竞争的产品**：寒(暑)假竞赛营、雏鹰杯专项营
- **竞品劣势**：
  - 实验室场地以及课程设计、宣传等方面**抄袭我们**，设备也不完善
  - 多为兼职外包带项目，**项目质量不可控**
  - 项目经验和赛事经验少，多局限在上海
  - 学料扩科的逻辑，多为鸡血拿奖，不注重能力培养
- **我方优势**：
  - **全职师资团队，教学经验丰富**：拥有极具特色的"交叉学科"教练团队，且均为全职。来匹配学生发展的不同需求。我们核心的教研团队，具备了 10 年的相关研发经验，均毕业于国内外名校
  - **区别于其他机构全职兼职，鱼龙混杂，多为大学生兼职带项目**：我们老师擅长将复杂的项目以及专业知识转化成学生能理解吸收的知识点，并且有体系化的基础课程，帮助学生构建科创思维
  - **坚持学生主导的项目原则**
- **退费条款对比**（销售敏感点）：
  - 昂立 0 获奖退 50% → **表面看起来"保一半"，但 50% 是沉没成本**
  - ideaLab 寒(暑)假竞赛营：小学 ¥49,800 = 项目孵化 ¥29,800 + 竞赛联报 ¥20,000，3 赛均未获奖退还申报费用一半即 ¥10,000
  - ideaLab 雏鹰杯专项营（2026-06-29 新方案）：¥29,800，两赛事均提交，未达起评奖级则退 50%
- **注意事项**：
  - 家长会拿退费条款比较，要解释清"零奖退一半"和"沉没成本"的关系
  - 不要直接说对方"抄袭"——避免法律风险，改用"借鉴/学习"
- **来源**：内部销售对比表（2026-07-01）
