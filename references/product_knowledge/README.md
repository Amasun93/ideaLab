# ideaLab课程产品结构化知识库

更新日期：2026-06-26

这是第二层数据清洗结果：它不是把 PPT/PDF 简单转成文字，而是把产品材料整理成 Agent 可用的产品知识库。

## Agent 入口

1. 先读 `SKILL.md`，确认回答边界和工作流程。
2. 再读 `产品总览.md`，理解产品家族、主线和高风险口径。
3. 需要推荐时读 `推荐逻辑.md` 和 `data/recommendation_rules.json`。
4. 需要讲具体产品时读 `产品卡片/<产品名>.md`。
5. 需要讲智造万物具体课节或项目时读 `课节索引/智造万物_课节速查卡.md` 和 `data/zhizao_wanwu_lesson_index.json`。
6. 需要找智造万物短视频/讲座素材时读 `素材索引/智造万物_本地素材索引.md`，再到本地素材库取文件。

## 文件结构

- `SKILL.md`：给其他 Agent 安装/加载后的工作说明。
- `产品总览.md`：产品体系全景、主线和使用原则。
- `产品关系图.md`：产品之间的衔接、升级、并行和专项关系。
- `产品卡片/`：每个产品一张结构化卡片。
- `课节索引/`：智造万物等完整课件入库后的课节级速查。
- `素材索引/`：本地素材库调用说明，只保存路径和使用规则。
- `销售与合同边界.md`：合同、退费、禁忌话术、对外表达边界。
- `推荐逻辑.md`：按家长目标和学生阶段的推荐规则。
- `口径冲突与待确认.md`：不同材料之间不一致的价格、课时、归属口径。
- `来源索引.md`：核心材料来源及其用途。
- `data/`：机器可读 JSON，用于后续检索、Agent装载或知识图谱导入。

## 使用边界

- 可以用于产品介绍、销售顾问辅助、课程路线推荐、材料检索、知识库导入。
- 不应用于直接生成最终报价单或合同承诺，报价和退费必须以最新销售 SOP/合同为准。
- 标为内部的材料不能直接转述给家长端。
- 本地素材库不上传 GitHub；公开使用学生照片、家长反馈和截图前要确认授权并打码。
