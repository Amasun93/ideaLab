# 口径冲突与待确认

以下记录产品素材中曾经存在的多口径，以及 2026-06-25 用户校准后的当前口径。Agent 使用时以“已校准口径”为准；仅对仍未确认的价格、合同、班型、材料缺口提醒回查当期销售 SOP。

## 已校准：MYO 命名体系

### 旧口径

- S01：MYO Basic / Standard / Pro，¥2,980 / ¥4,980 / ¥7,980。
- S02：3D打印基础营、3D打印进阶营、3D打印卡丁车制造营、青少年科创发明营。

### 当前口径

- MYO 对外使用主题营命名：3D打印基础营、3D打印进阶营、3D打印卡丁车制造营。
- 青少年科创发明营不是 MYO；它与科创发明营是同一个产品。

### Agent 动作

- 介绍 MYO 时只讲主题营，不把青少年科创发明营放入 MYO。

## 已校准：科创发明营归属

### 旧口径

- AI 小芮稿：科创发明营面向中学生，价格详询校区。
- 产品体系稿：青少年科创发明营放在 MYO 表中，价格 ¥49,800-¥54,800。

### 当前口径

- 青少年科创发明营和科创发明营是同一个产品。
- 科创发明营不是 MYO，也不是寒(暑)假竞赛营。
- 科创发明营是“不打比赛的科创竞赛营”：7 天课内训练内容与寒(暑)假竞赛营相近，但没有后续比赛申报和材料准备环节。
- 交付为作品/项目过程素材和约 1000 字左右项目说明；不交付正式赛事论文和比赛材料。

### Agent 动作

- 不把科创发明营推荐给明确要比赛申报、论文、比赛材料或获奖退费承诺的家庭。

## 已校准：寒(暑)假竞赛营命名

### 旧口径

- 材料中同时出现竞赛集训营、Lab赛事包、赛事包、寒暑假科创竞赛集训、连报等表达。

### 当前口径

- 对外统一叫寒(暑)假竞赛营。
- 竞赛集训营和 Lab赛事包都是这个产品的旧称/别名。

### Agent 动作

- 面向家长优先说寒(暑)假竞赛营；解释旧材料时可说“也就是以前材料里的竞赛集训营 / Lab赛事包”。

## 已校准：科创启航计划价格与课时

### 旧口径

- S02：科创启航计划 ¥20,000，73 课时。
- S01：启航 1.0 48 课时 ¥19,800；启航 2.0 72 课时 ¥28,800。
- S12：正式课设计资料出现 15 次课 45 课时 + 3 天集训 28 课时等教学设计口径。

### 当前口径

- 当前在售口径：¥20,000 / 73 课时。
- 73 课时 = 春/秋周课 45 课时 + 假期集训 28 课时。
- 1 课时 = 45 分钟。
- 启航不区分 1.0 和 2.0。

### Agent 动作

- 可以直接使用 ¥20,000 / 73 课时口径；退费、排班和合同细节仍以当期 SOP/合同为准。

## 已校准：科创领航计划价格、课时与入门测

### 旧口径

- S02：科创领航计划 ¥29,800，84 课时 / 63 小时。
- S01：领航计划 96 课时，¥45,800。

### 当前口径

- 当前在售口径：¥21,800 / 73 课时。
- 73 课时 = 春/秋周课 45 课时 + 假期集训 28 课时。
- 1 课时 = 45 分钟。
- 非在读学员必须参加领航入门测。

### Agent 动作

- 可以直接使用 ¥21,800 / 73 课时口径；遇到非在读学员报名，必须提醒先做入门测。

## 已校准：智造万物课时

### 当前口径

- 每阶 80 课时/年，约 40 课次/年。
- 每次 2 小时。

### Agent 动作

- 对外使用“每阶约 40 次课 / 80 课时，每次 2 小时”。
- 清洗版里的 Lesson 标记用于内部课纲拆解，不作为对外课次数。

## 已校准：未来工程师专项营退费

### 当前口径

- 售价 ¥15,000。
- 区赛未获奖退费 50%，即 ¥7,500。
- 家长自行放弃参赛不退。
- 行政区未办区赛但项目推送市赛，市赛未获奖也不退。

## 已校准：雏鹰杯专项营价格、申报与退费

### 当前口径

- 售价 ¥29,800。
- 同时投递雏鹰杯和长三角两个赛事。
- 雏鹰杯获得起始奖项或长三角获得起始奖项均不退费。
- 例：雏鹰杯获得区奖，或长三角获得市奖，都不退费。
- 两个比赛都没有获得起始奖项，退费 50%，即 ¥14,900。

## 已校准：高中课题营归属

### 当前口径

- 高中课题营可以理解为启航计划、领航计划的高中版。
- 启航偏小学，领航偏初中，高中课题营偏高中。
- 通过寒假或暑假 7 天集训完成一个高中综评课题。
- 不打比赛，服务高考综评准备。

### Agent 动作

- 不把高中课题营推荐成赛事服务产品。
- 家长要比赛申报时，转向寒(暑)假竞赛营或专项营。

## 已校准：赛事服务禁忌与推荐表达

### 当前口径

- 所有赛事服务统一禁止说“保奖 / 保证获奖 / 100% 拿奖”。
- 推荐表达：比赛获奖由多种因素决定。我们能做的是基于多年经验，找到主流评委认可的一些方向和选题，避免踩雷的高风险选题；这个过程还需要家长、学校老师和学生配合，最终也受赛事评审标准和评委偏好影响。

## 已校准：竞赛营战报数据

### 当前口径

- 竞赛营战报数据可以使用。
- 使用时必须标明为机构历史服务结果/产品宣贯数据，不是官方统计，也不是未来获奖承诺。
