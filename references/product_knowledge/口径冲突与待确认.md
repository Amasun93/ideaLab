# 口径冲突与待确认

以下记录产品素材中曾经存在的多口径，以及 2026-06-25 至 2026-06-26 校准后的当前口径。Agent 使用时以“已校准口径”为准；仅对仍未确认的价格、合同、班型、材料缺口提醒回查当期销售 SOP。

## 已校准：MYO 命名体系

### 旧口径

- S01：MYO Basic / Standard / Pro，¥2,980 / ¥4,980 / ¥7,980。
- S02：3D打印基础营、3D打印进阶营、3D打印卡丁车制造营、青少年科创发明营。

### 当前口径

- MYO 对外使用主题营命名：3D打印基础营、3D打印进阶营、3D打印卡丁车制造营。
- 青少年科创发明营不是 MYO；它与科创发明营是同一个产品。

### Agent 动作

- 介绍 MYO 时只讲主题营，不把青少年科创发明营放入 MYO。

## 已校准：科创发明营归属

### 旧口径

- AI 小芮稿：科创发明营面向中学生，价格详询校区。
- 产品体系稿：青少年科创发明营放在 MYO 表中，价格 ¥49,800-¥54,800。

### 当前口径

- 青少年科创发明营和科创发明营是同一个产品。
- 科创发明营不是 MYO，也不是寒(暑)假竞赛营。
- 科创发明营是“不打比赛的科创竞赛营”：7 天课内训练内容与寒(暑)假竞赛营相近，但没有后续比赛申报和材料准备环节。
- 交付为作品/项目过程素材和约 1000 字左右项目说明；不交付正式赛事论文和比赛材料。

### Agent 动作

- 不把科创发明营推荐给明确要比赛申报、论文、比赛材料或获奖退费承诺的家庭。

## 已校准：寒(暑)假竞赛营命名

### 旧口径

- 材料中同时出现竞赛集训营、Lab赛事包、赛事包、寒暑假科创竞赛集训、连报等表达。

### 当前口径

- 对外统一叫寒(暑)假竞赛营。
- 竞赛集训营和 Lab赛事包都是这个产品的旧称/别名。

### Agent 动作

- 面向家长优先说寒(暑)假竞赛营；解释旧材料时可说“也就是以前材料里的竞赛集训营 / Lab赛事包”。

## 已校准：寒(暑)假竞赛营费用拆分、联报赛事与退费

### 当前口径

- 小学竞赛营：总价 ¥49,800。项目孵化课程部分 ¥29,800；竞赛联报服务 ¥20,000，小学 3 赛，包含青创赛（区级）、雏鹰杯（区级起）、宋庆龄少年儿童发明奖（市级起）的申报以及晋级后的答辩辅导服务。若 3 个比赛均未获奖，退还申报费用的一半，即 ¥10,000。
- 初中竞赛营：总价 ¥54,800。项目孵化课程部分 ¥29,800；竞赛联报服务 ¥25,000，初中 4 赛，包含青创赛（市级起）、明日科技之星/明科（区级起）、宋庆龄少年儿童发明奖（市级起）、雏鹰杯（区级起）的申报以及晋级后的答辩辅导服务。若 4 个比赛在周期内均未获奖，退还申报费用的一半，即 ¥12,500。
- 高中竞赛营：总价 ¥49,800。项目孵化课程部分 ¥29,800；竞赛联报服务 ¥20,000，高中 3 赛，包含青创赛（市级起）、明日科技之星/明科（区级起）、宋庆龄少年儿童发明奖（市级起）的申报以及晋级后的答辩辅导服务。若 3 个比赛在周期内均未获奖，退还申报费用的一半，即 ¥10,000。
- 项目孵化课程部分包含开题定题、项目设计、项目落地与授课指导、过程性素材采集；不包含专业论文。
- 交付项目原型以及对应比赛申报材料，按比赛要求准备论文/研究报告、申报表、查新报告、研究日志、演讲视频、PPT/答辩材料等。

### Agent 动作

- 价格问题可直接按小学/初中/高中三档回答，但必须提醒具体排班、合同和退费以校区签约为准。
- 不把“全部联报赛事均未获奖退申报费一半”说成保奖或结果保证。
- 家长问“为什么课程部分不含专业论文”时，要区分项目孵化费和竞赛联报服务：课程孵化不单独承诺专业论文写作，竞赛申报材料会按对应赛事要求整理。

## 已校准：科创启航计划价格、课时与招募年龄

### 旧口径

- S02：科创启航计划 ¥20,000，73 课时。
- S01：启航 1.0 48 课时 ¥19,800；启航 2.0 72 课时 ¥28,800。
- S12：正式课设计资料出现 15 次课 45 课时 + 3 天集训 28 课时等教学设计口径。

### 当前口径

- 当前在售口径：¥20,000 / 73 课时。
- 73 课时 =（春季15次周课45课时 + 暑假集训28课时（3天，每天7小时））或（秋季15次周课45课时 + 寒假集训28课时（3天，每天7小时））；每次周课2.5小时，集训3天每天7小时。
- 1 课时 = 45 分钟。
- 启航不区分 1.0 和 2.0。
- 上海区域最小招募 9 月开学三年级，即暑假 2 升 3 可招募；暑假 1 升 2 不再新招，已招募学员继续服务。

### Agent 动作

- 可以直接使用 ¥20,000 / 73 课时口径；先按 9 月开学年级筛选；退费、排班和合同细节仍以当期 SOP/合同为准。

## 已校准：科创领航计划价格、课时、年龄与入门测

### 旧口径

- S02：科创领航计划 ¥29,800，84 课时 / 63 小时。
- S01：领航计划 96 课时，¥45,800。

### 当前口径

- 当前标准价口径：¥21,800 / 73 课时。
- 73 课时 =（春季15次周课45课时 + 暑假集训28课时（3天，每天7小时））或（秋季15次周课45课时 + 寒假集训28课时（3天，每天7小时））；每次周课2.5小时，集训3天每天7小时。
- 1 课时 = 45 分钟。
- 9 月开学六年级及以上才进入领航沟通。
- PY3/PY4 在读学员或参加过启航计划的学员，符合年龄要求时可免测报名，优惠后为 ¥19,800。
- 非在读学员必须参加并通过领航入门测；未通过禁收领航，可转向启航或其他更适配产品。

### Agent 动作

- 可以直接使用标准价 ¥21,800 / 73 课时口径；先筛 9 月开学年级和是否 PY3/PY4 在读/启航学员；遇到非在读学员报名，必须提醒先做入门测。

## 已校准：高中课题营归属

### 当前口径

- 高中课题营可以理解为启航计划、领航计划的高中版。
- 启航偏小学，领航偏初中，高中课题营偏高中。
- 通过寒假或暑假 7 天集训完成一个高中综评课题。
- 不打比赛，服务高考综评准备。

### Agent 动作

- 不把高中课题营推荐成赛事服务产品。
- 家长要比赛申报时，转向寒(暑)假竞赛营或专项营。

## 已校准：赛事服务禁忌与推荐表达

### 当前口径

- 所有赛事服务统一禁止说“保奖 / 保证获奖 / 100% 拿奖”。
- 推荐表达：比赛获奖由多种因素决定。我们能做的是基于多年经验，找到主流评委认可的一些方向和选题，避免踩雷的高风险选题；这个过程还需要家长、学校老师和学生配合，最终也受赛事评审标准和评委偏好影响。

## 已校准：竞赛营战报数据
