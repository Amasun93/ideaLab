# MYO系列

- 产品ID：`myo`
- 类型：regular_course
- 分类：常规课程产品
- 定位：短期项目制课程，让学生在较短时间内完成可见作品；也可作为深入课程前的体验入口。
- 目标学员：小学阶段为主；部分材料写作各年级或2年级及以上。
- 命名口径：对外使用主题营命名，即 `3D打印基础营 / 3D打印进阶营 / 3D打印卡丁车制造营`，不再优先使用 MYO Basic / Standard / Pro 三档口径。
- 价格口径：以当期销售 SOP 为准；历史材料中的 MYO Basic / Standard / Pro 仅作为旧口径保留，不作为当前优先介绍方式。
- 课时口径：按具体主题营确认；当前资料中 3D 打印基础营、进阶营为 5 天 x 3 小时，卡丁车制造营为 4 天 x 3 小时。
- 合同类型：常规学员合同。

## 核心交付

- 实物作品
- 短期PBL体验
- 3D打印机开箱安装、切片、打印、排错与基础维护体验
- 3D打印/建模/制作体验
- 真实场景修复与工程大项目体验

## 适合推荐给谁

- 先体验再决定是否深入
- 需要低门槛可见作品
- 暑期/短期科技主题课程

## 不适合或要谨慎推荐

- 需要论文产出
- 需要明确赛事申报服务

## 版本口径

- 当前对外优先使用主题营口径：3D打印基础营、3D打印进阶营、3D打印卡丁车制造营。
- MYO Basic / Standard / Pro 是旧材料中的三档口径，不作为当前优先销售命名。
- 青少年科创发明营不是 MYO；它与科创发明营是同一个产品，应单独归入科创发明营资料卡。

## 口播介绍素材

- 一句话定位：MYO 是 Make Your Own 的短期项目制课程，把前沿技术做成孩子能亲手完成、能展示、能带走的作品。
- Robin 式开场可用方向：介绍给想先体验科技项目、先看孩子动手能力和兴趣点的家庭。
- 口播重点：不用讲成长期路线，重点讲清楚每个营孩子做什么、带走什么、家长能看到什么；基础营不要写成普通建模课，重点是孩子能独立使用3D打印工具并理解试错排错。
- 年级口径：MYO 主题营目前主要面向 2 年级及以上孩子；基础营、进阶营、卡丁车营的单条口播均应在开头说明适合年级。
- 产品边界：MYO 是短期 PBL 和作品体验入口，不默认包含论文产出、赛事申报或获奖服务。

## 主题营结构

### 未来造物者・3D 打印基础营

- 适合年级：2 年级及以上。
- 课程安排：5 天，每天 3 小时。
- 课程主线：从制造技术和工业发展史出发，认识 3D 打印原理与应用，组装一台 3D 打印机，打印第一个模型，再学习模型下载、切片、参数设置、打印试错、上色、打磨和简单建模。
- 孩子具体做什么：参与开箱、安装和调试 3D 打印机；认识 PLA/PETG/ABS 等材料特性；使用 Maker World 下载和导入模型；修改模型满足自己的需求；学习切片软件和打印参数；经历支撑、翘边、尺寸不合适等常见失败的判断和调整；学习三视图和基础建模；做一个融合 3D 打印结构件与电子元件的玩具小车。
- 学生成果：自主安装一台 3D 打印机，打印第一个模型；完成自主下载/修改/打印模型；完成小车产品模型及可动成品；最终形成实物作品、证书和演讲视频记录。
- 家长看点：这不是只打印一个小玩具，也不是单纯学建模功能，而是让孩子完整经历“认识工具-操作设备-寻找模型-切片参数-打印试错-排查维护-设计作品-公开展示”的短项目流程。

### 未来造物者・3D 打印进阶营

- 适合年级：2 年级及以上，但更适合会使用 3D 打印机和切片软件的学生，或基础营与进阶营连报的新学员。
- 课程安排：5 天，每天 3 小时。
- 课程主线：3D 打印 x AI x 电子，夯实建模与打印功底，进阶学习结构测量、精密修复、AI 辅助创意、简易交互功能，并引入 3Dfiti / 城市修复概念。
- 孩子具体做什么：练习测量三法则、布尔运算、尺寸公差、卡扣拼接；体验 AI 图像 2D 转 3D、参数化建模、壁厚与圆角优化；设计磁吸结构和 AI 故事角色；通过用户研究发现城市公共设施或日常物件的修复需求，完成带简易交互功能的城市修复半成品。
- 学生成果：创意小摆件、Gridfinity 收纳盒或异形工具槽、个性化磁性冰箱贴、故事角色小模型、带简易交互功能的城市修复半成品、完整作品集。
- 家长看点：进阶营的价值不只是会建模，而是让孩子把 AI、测量、结构、电子和真实公共场景放在一个项目里使用，训练同理心、用户研究和工程迭代；城市修复作品可作为校内展示或小型项目展示素材，但不要承诺比赛结果。

### 3D 打印卡丁车制造营

- 适合年级：2 年级及以上。
- 课程安排：4 天，每天 3 小时。
- 课程主线：3D 打印 + 汽车工程 + DIY 拼装，让孩子从 0 到 1 组装一台可驾驶、可带回家的卡丁车；该方案强调自研、可规模化落地和工程体验，不应写成普通玩具拼装。
- 孩子具体做什么：学习项目管理和安全守则；认识车辆历史、卡丁车结构、工具和紧固件；搭建车架；用 TinkerCAD 建模设计开关座等外观配件；搭建动力系统、座椅、电路和转向系统；使用万用表做电路检测；完成试驾、测试、团队比赛和项目汇报。
- 学生成果：车辆底盘、3D 打印外观件、动力系统、座椅、转向系统、卡丁车成品、项目笔记和项目汇报。
- 家长看点：这门课的吸引力是“大玩具”，但真正的课程价值是机械结构、工业设计、电路连接、阿克曼转向、测试优化和项目表达。
- 大卫老师补充口径（待证据补充后可对外强化）：该卡丁车方案为自研 3D 打印卡丁车项目，曾有对外展出/展车素材；如要在公开视频中使用“国内唯一量产方案”“深圳 Maker Faire 最佳展品”“拓竹深圳官方旗舰店展车”等强表述，应先补图片、链接或证据材料。

### 青少年科创发明营边界

- 青少年科创发明营不是 MYO。

- 写 MYO 常规课程介绍时，不要把科创发明营放进 MYO 低门槛体验产品中。
