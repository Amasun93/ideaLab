# 科创发明营（纯项目 / 纯项目科创营）

- 产品ID：`kechuang_invention_camp`
- 类型：project_camp
- 分类：短期科创项目营/非赛事申报产品
- **核心定位：竞赛营的"纯项目"版本。** 科创发明营 = 寒(暑)假竞赛营的项目孵化部分（¥29,800），**只做项目，不做比赛申报。** 课程内容、7天集训、项目交付与竞赛营完全一致，但不包含后续的竞赛联报服务、论文优化、赛事申报材料和答辩辅导。
- **与竞赛营的关系**：竞赛营 = 科创发明营（纯项目 ¥29,800）+ 竞赛联报服务（¥20,000/¥25,000）。孩子上完科创发明营如果觉得项目不错，可以再补竞赛联报服务升级为竞赛营。
- 目标学员：需要集中完成一个原创科创发明项目，但暂不走赛事申报的学生；希望先做项目、后续再决定是否冲比赛的家庭。
- 价格口径：已校准为 **¥29,800**（纯项目）。
- 在售状态：✅ 当前可售（2026-07-01 用户确认）。
- 课时口径：7 天集训，与寒(暑)假竞赛营课内训练完全一致；具体课时以当期班型确认。
- 合同类型：非赛事申报产品，合同与退费按当期销售 SOP 和正式合同确认。

## 一句话介绍

科创发明营就是"不报比赛的竞赛营"：用 7 天完成一个原创科创项目，课程内容、项目质量和训练强度与竞赛营一模一样，但不包含后续的赛事申报。孩子先做出好项目，后面想冲比赛了再补竞赛联报服务就行。

## 课程亮点

- **真实问题驱动**：从生活中发现真实问题，不是老师灌题目
- **原创项目制作**：每个孩子完成一个个性化原创发明项目
- **项目全流程陪跑**：从定题到成品，老师全程带
- **能力全程提升**：动手能力、工程思维、问题解决能力都在项目中提升

## 项目路径（5步走）

1. **发现问题** → 从日常生活/社会观察中识别可解决的问题
2. **方案设计** → 需求分析、技术方案论证、系统设计
3. **动手制作** → 元器件认识、组装接线、编程调试
4. **实验测试** → 测试数据采集、结果分析、迭代优化
5. **成果整理** → 项目装置完成 + 过程素材 + 1000字项目说明

## 核心交付

- **1个个性化原创项目装置**（实物作品，孩子亲手参与）
- **学生过程素材**（制作日志、测试数据、照片/视频记录）
- **约1000字项目说明**（孩子对项目的理解和表达）
- 不交付正式赛事论文、查新报告、答辩PPT或比赛申报材料

## 适合推荐给谁

**通用画像：**
- 想做一个有质量的原创科创项目的学生（小学到高中均可）
- 对科学/AI/工程感兴趣、喜欢动手创造的学生
- 希望积累项目经历、但还不确定要不要走竞赛路线的家庭

**科创发明营（¥29,800）暑期特别适配画像：**
- 孩子大部分时间在国外、只有暑假回国，想集中做一个个性化项目
- 平时没有时间上启航等长期课程，想暑假集中体验个性化项目制作
- 国际学校学生，平时时间不好安排，暑假来做个项目

**竞品参考：** Wowkids 做的是 3 个人团队的团队项目，收费 ¥29,800/人；科创发明营是**一对一的个性化项目**，同样 ¥29,800，性价比更高。

## 暑假竞赛营销售优先级

**原则：暑假竞赛营席位优先收 ¥49,800/¥54,800 的竞赛营学生。** 但如果遇到符合科创发明营画像的学生（海外回国、国际学校、时间紧张想做项目），**可以正常签约 ¥29,800**，不需要硬推全套竞赛服务。

## 与竞赛营的选择关系（重要！）

**科创发明营和竞赛营是报名时二选一的关系，不是"先上再补差价升级"。** 报名时就决定：
- 选择**科创发明营 ¥29,800** → 只做项目，不参加比赛
- 选择**竞赛营 ¥49,800/¥54,800** → 做项目 + 全套赛事申报服务

如果孩子做完科创发明营后想冲比赛，需要单独做赛事规划，但不是直接补差价升级。具体以当期销售 SOP 为准。

## 与领航计划的区别（重要！）

| 维度 | 科创发明营（¥29,800） | 领航计划（¥21,800） |
|------|----------------------|---------------------|
| 产品类型 | 7天短期集训 | 长期课程（73课时，春季15次周课+暑假集训3天 或 秋季15次周课+寒假集训3天） |
| 核心目标 | 集中完成一个原创科创项目 | 深度科研训练+科研论文 |
| 项目深度 | 做一个完整的项目装置 | 从观察到数据论证的完整科研闭环 |
| 论文 | 约1000字项目说明 | 结构化科研论文 |
| 数据论证 | 有测试优化，但非核心要求 | 核心要求：实验设计+数据分析+论证 |
| 入门要求 | 无门槛 | 需要入门测试（非在读学生） |
| 赛事出口 | **不参加比赛（纯项目）** | **参加ICC + 长三角2个比赛** |
| 适合谁 | 想短周期出项目作品 | 想做长期深度科研 + 有赛事出口 |

> 简单说：科创发明营是"用7天做一个好项目"，领航是"用73课时学会怎么做科研"。
