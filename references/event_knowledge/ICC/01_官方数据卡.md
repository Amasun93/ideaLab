# ICC 官方数据卡

## 赛事身份

| 字段 | 内容 |
|------|------|
| 全称 | 全球发明大会中国区（Invention Convention China, ICC） |
| 全球母赛事 | ICW（Invention Convention Worldwide），美国亨利·福特博物馆举办 |
| 中国区主办方 | 中国人民对外友好协会 → 中国友好和平发展基金会 |
| 教育部白名单 | ✅ 2022-2025 学年面向中小学生的全国性竞赛活动 |
| 创办/获批年份 | 2022 年正式获教育部批准 |
| 国际覆盖 | 全球 K-12 年级学生（ICW） |

## 参赛规则

| 字段 | 内容 |
|------|------|
| 参赛对象 | 全国小学、初中、高中、中专和职高学生 |
| 参赛形式 | 单人或组队（≤4人）；不可跨省组队；同一队员仅可参加一个团队 |
| 组别 | G组（1-2年级）/ P组（3-5年级）/ J组（6-9年级）/ S组（10-12年级） |
| 组别判定 | 由团队中学段最高的队员决定 |
| 竞赛主题 | 15 个主题，结合十四五规划和联合国 17 个可持续发展目标 |

## 评审流程

| 阶段 | 内容 | 时间 |
|------|------|------|
| 作品合规审查 | 材料完整性 + 技术可行性 | 省赛现场评审前 15 天 |
| 预先评审 | 评委审阅材料并打分 | 现场评审前 7 天 |
| 现场评审 | 结合预评得分 + 现场表现 | 省赛 / 国赛现场 |
