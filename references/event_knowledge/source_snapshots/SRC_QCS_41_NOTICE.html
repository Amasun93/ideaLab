【赛复活动】关于举办第41届上海市青少年科技创新大赛的通知

2734 阅读 0 评论 0 点赞


    由上海市科学技术协会、上海市教育委员会、上海市科学技术委员会、上海市生态环境局、上海市绿化和市容管理局、上海市知识产权局、共青团上海市委员会、上海市妇女联合会、上海市气象局、中国福利会、中国科学院上海分院、上海科技馆、、上海科技发展基金会、上海市科普基金会共同主办的第41届上海市青少年科技创新大赛，将于2026年1月至4月举办。根据《第41届上海市青少年科技创新大赛实施方案》（沪科青创委〔2025〕5号），现就有关事项通知如下：


一、大赛内容


    第41届上海市青少年科技创新大赛分为青少年科技创新竞赛板块和少年儿童科学幻想绘画展示板块。按照全国大赛名额分配、年龄和性别等要求，择优推荐学生参加第40届全国青少年科技创新大赛。


二、时间安排

    第41届上海市青少年科技创新大赛分四个阶段组织实施。

第一阶段（作品申报）

    2026年1月1日8:00至1月20日24:00为大赛作品申报阶段，凡符合规定的申报者均可登录大赛官方网站（网址：https://www.shssp.org.cn）进行自主申报。申报者须在规定时间内完成在线申报，逾期视为放弃。具体申报规则见附件1、附件2。

第二阶段（线上初审）

    2026年1月下旬至3月上旬为资格审查、原创和伦理审查以及专家初评阶段。对参赛学生的申报材料进行资格审查，并对作品的原创性以及是否遵守科学研究道德和科技伦理规范、行为准则等进行审查。资格审查以及原创和伦理审查合格，方可进入专家初评。

    青少年科技创新竞赛板块专家初评，将按学科和年龄组对申报作品的研究创新水平、研究日志及实验报告等进行评审；根据初评专家建议和随机抽取的方式，对部分作品进行在线问辩。根据专家初评成绩，以及进入初评的各年龄组和各学科作品数量，按比例选拔不超过400个作品的参赛学生进入现场活动。

    少年儿童科学幻想绘画展示板块专家初评，对申报材料的科学性、艺术性、创意性等进行评审，根据专家初评成绩，按申报总数60%的比例选拔参赛学生进入现场活动。

第三阶段（现场活动）

    3月中旬参赛学生可登录报名网站查询线上初审结果。

    3月下旬在上海科学会堂举行青少年科技创新竞赛板块作品现场问辩、动手实践考察、创新素养和综合素质考察，以及少年儿童科学幻想绘画展示板块技能测试、展示分享等现场活动。现场活动相关安排和要求另行通知。

第四阶段 （结果公布）

    4月对大赛获奖名单进行公示并公布结果；根据第40届全国青少年科技创新大赛安排适时公示、确定推荐参赛名单。

    本赛事不纳入上海市中学生综合素质评价。
