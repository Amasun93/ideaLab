# IEYI 官方数据卡

## 赛事身份

| 字段 | 内容 |
|------|------|
| 全称 | International Exhibition for Young Inventors（IEYI） |
| 中文名 | 世界青少年创新发明展 / 世界青少年创新发明大会 |
| 创办年份 | 2004 年 |
| 成员国/地区 | 近 40 个 |
| 中国加入年份 | 2016 年 |
| 报名方式 | 通过 IEYI 中国区认证合作伙伴统一组织 |
| 教育部白名单 | 推文中未明确提及白名单身份 |

## 参赛规则

| 字段 | 内容 |
|------|------|
| 参赛对象 | 小学 1 年级至中学 12 年级 |
| 组别判定 | 团队参赛时以年龄最大的成员为准 |
| 参赛类别 | 创新发明制作类 + 创意艺术绘画类 |

## 创新发明制作类主题

| 序号 | 主题 |
|------|------|
| 1 | 灾害管理 / 安全与健康 |
| 2 | 娱乐与体育 / 教育 |
| 3 | 可持续农业科技 |
| 4 | 社会关怀 |
| 5 | 便利生活 |
| 6 | 愿景技术（绿色环保 + 航空航天/AI/IoT） |

## 评审维度

### 创新发明制作类

| 维度 | 权重 |
|------|------|
| 创意性 | 40% |
| 市场效益性 | 30% |
| 操作性 | 30% |

资格否决项：科学原理实验、纯艺术创作、具有危险性 → 直接丧失参评资格

### 创意艺术绘画类

| 维度 | 权重 |
|------|------|
| 综合评估 | 40% |
| 创造力 | 30% |
| 构图 | 30% |
