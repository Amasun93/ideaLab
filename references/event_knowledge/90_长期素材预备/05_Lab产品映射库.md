# Lab产品映射库

更新时间：2026-06-11

用途：把赛事要求转成 Lab 课程产品语言。Agent 回答时应从“赛事百科”自然落到“我们如何交付”。

|赛事要求|Lab模块|交付物|Agent提示|
|---|---|---|---|
|真实问题发现|选题诊断与问题重构|问题树、用户/场景分析、一句话项目说明|先判断问题是否真实、是否可验证、是否适合孩子年级。|
|原创性与查新|查新与竞品/同类方案调研|查新清单、同类方案表、创新点边界|不要夸大创新，明确已有方案和本项目差异。|
|研究过程和数据|研究设计与过程管理|研究日志、实验记录、数据表、迭代记录|强调孩子参与过程，避免只有最终成品。|
|实物/模型/程序|原型搭建与作品拍摄|作品原型、关键结构图、功能演示照片/视频|把作品功能拆成可演示的3到5个证据点。|
|报告和申报材料|赛事材料打磨|研究报告、申报表、作品摘要、材料清单|按赛事当届规则适配格式，不复用错赛道模板。|
|现场问辩/表达|答辩表达训练|答辩稿、问答库、3分钟讲解、评委追问模拟|训练孩子讲清楚：问题从哪来、怎么做、为什么有效、下一步改什么。|
|三赛联动|主赛+备赛矩阵|赛事匹配表、复用材料清单、申报节奏表|同一项目可不同表达，但不能混淆不同赛事规则和分母。|
