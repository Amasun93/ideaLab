# 家长问答库

更新时间：2026-06-11

用途：给 WorkBuddy、社群运营和销售顾问回答常见问题。回答要稳，不承诺保奖。

|问题|建议回答|风险提醒|
|---|---|---|
|三项赛事是不是都要参加？|不一定。先看孩子项目类型、年级、研究深度和表达状态，再决定主赛和备赛。Lab赛事包的价值是做赛事矩阵，不是盲目多报。|不能承诺多报就一定获奖。|
|没有实物作品能不能做？|可以先做选题诊断，但不同赛事对实物、模型、报告、展示材料要求不同。青创赛更看研究过程和创新素养，宋庆龄发明作品强调完整实物，创意/AI编程类别要求又不同。|要根据当届规则和赛道判断。|
|几月开始准备比较合适？|建议至少提前3到6个月。青创赛通常年初申报、春季评审；宋庆龄发明奖通常上半年申报初评、暑期终评；雏鹰杯通常秋季启动、冬季/次年初终评。|具体日期每年会变，发布前查当届通知。|
|孩子低年级适合冲哪个？|低年级更适合从真实生活问题和可展示的小原型开始，优先判断雏鹰杯/青创赛低龄组/科幻画等入口；宋庆龄发明奖的发明、AI编程、创意作品小学组通常要求四年级及以上。|年级限制以当届通知为准。|
|青创赛对综评有没有用？|要讲准确：第41届青创赛官方通知明确本赛事不纳入上海市中学生综合素质评价。可以说项目研究、表达训练和赛事经历本身有能力培养价值，但不能承诺综评或升学结果。|不能把参赛或获奖说成综评承诺。|
|雏鹰杯和小院士是什么关系？|雏鹰杯是上海少先队系统下的科创展示和筛选路径之一，和上海少年科学院“小院士”交流展示紧密相关。2025年度公开报道显示，按项目类别取前3%入围第23届小院士终评，最终10位少先队员获“小院士”称号。|当前最新口径来自媒体/政务号公开报道，后续如有官方原文应优先替换。|
|今年宋庆龄发明奖什么时候准备？|第二十一届官方通知显示：2026年2-3月各地选拔推荐，3-4月申报及形式审核，5-6月全国初评，6月/7月入围名单公示，8月终评颁奖。|具体报送入口、地区截止日期和名额以各组织机构通知为准。|
|查新/查重报告重要吗？|重要。赛事越来越重视原创性和学生真实参与，查新/查重不是形式，而是帮孩子知道自己的创新点在哪里。|不能代做或伪造数据。|
|为什么老师要反复改报告和答辩？|评委看的不是漂亮话，而是孩子是否理解问题、方法、实验过程、数据和局限。报告和答辩训练是把项目从“做出来”变成“讲清楚”。|不要把辅导讲成包办。|
|获奖率能不能直接对比机构水平？|不能简单对比。不同机构的样本、奖项层级、参赛地区、统计分母都不同。Lab内部更关注同一团队在三赛矩阵里的稳定交付。|不要点名友商对比。|
|学生照片、证书、展板能不能直接发？|内部复盘可以留档，公开发布前必须确认授权，并尽量弱化学生隐私信息。|图片库必须做授权分级。|
