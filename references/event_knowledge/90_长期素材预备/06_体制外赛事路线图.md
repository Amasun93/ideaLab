# 体制外（国际教育方向）赛事路线图

更新时间：2026-06-28

> 本文件面向校区顾问和内容团队，为计划走国际教育路线（双轨制/国际学校/海外升学）的学生家庭提供赛事规划和产品匹配参考。

---

## 一、核心逻辑：体制外看什么

体制外赛事和体制内综评赛事的核心差异：

| 维度 | 体制内 | 体制外 |
|------|--------|--------|
| 看重什么 | 综评加分、升学指标 | 学术证明力、差异化竞争力 |
| 赛事导向 | 白名单/官方赛事 | 国际认可度、名校招生办是否认可 |
| 项目风格 | 符合国内评审偏好 | 注重个人兴趣、社会影响力、跨学科 |
| 关键差异 | 获奖结果更重要 | **参与过程、成长轨迹、项目叙事同样重要** |

**一句话：** 体制外申请不是"拿一个奖就能上"，而是通过赛事证明孩子的学术热情、探索能力和坚持力。

## 四、与 ideaLab 产品的交叉

### 4.1 竞赛营 → ISEF 通路

**这是体制外家庭最大的产品价值点：**

> 竞赛营联报青创赛（市级起） → 青创赛优秀项目可被推荐参加 ISEF（国际科学与工程大奖赛）

ISEF 被称为"科创界的诺贝尔奖"，是体制外申请中**最顶级的学术证明之一**。这个通路对于目标藤校/G5的家庭非常有说服力。

**顾问话术示例：**
> "竞赛营的项目孵化 ¥29,800 做的是原创科研项目，竞赛联报 ¥20,000 对接的不仅是国内的青创赛、明科、宋赛——青创赛作为 ISEF 的中国区选拔通道，孩子如果项目足够优秀，是有机会走上国际舞台的。这条线在体制外申请里面价值很高。"

### 4.2 智造万物 → 机器人赛事基础

智造万物的工程搭建、3D 打印、编程基础，对应：
- FLL Challenge 的基础技能
- VEX IQ/VRC 的工程思维
- WRO 的结构设计能力

### 4.3 MYO → 海外申请作品集素材

MYO 项目产出的实物作品和过程记录，可直接用于体制外申请的**项目展示（portfolio）**，体现孩子的动手能力、创造力和坚持力。

### 4.4 未来工程师 → 综评与国际接轨

未来工程师的航天实验装置设计，对体制外学生而言不仅是比赛经历，更是**工程类项目作品**：

> 海外招生办看重的是：孩子从发现问题 → 设计方案 → 动手制作 → 测试优化 → 呈现表达的完整闭环。未来工程师正好覆盖了这个链条。

## 五、各阶段推荐组合

### 小学低年级（G1-G3）— 兴趣探索期

| 方向 | 推荐组合 | 产品匹配 |
|------|---------|---------|
| 综合打底 | 袋鼠数学 + Bebras + FLL Discover | MYO/智造万物 |
| 工程方向 | VEX IQ + Scratch国际赛 | 智造万物 |
| 科学方向 | Science Olympiad 小学版 | MYO |

> **原则**：不急于出成绩，重在让孩子感受"做项目"和"团队协作"的乐趣。竞赛营不推荐这个年龄段。

### 小学高年级（G4-G5）— 积累期

| 方向 | 推荐组合 | 产品匹配 |
|------|---------|---------|
| 数学 + 科创 | AMC 8 + FLL Challenge + 科创比赛试水 | **科创启航计划** → 积累原创作品，衔接G6+竞赛营 |
| 编程方向 | Bebras + USACO青铜 + VEX IQ | 智造万物进阶 |
| 工程方向 | VEX IQ/VRC + WRO | 智造万物 / 3D打印机营 |

> **原则**：开始有目标地积累奖项和经验。启航计划（G3+可入）可以帮助孩子建立正式的项目能力和答辩基础。

### 初中（G6-G8）— 冲刺期

| 方向 | 推荐组合 | 产品匹配 |
|------|---------|---------|
| 科研方向 | **竞赛营（青创赛→ISEF通路）** + AMC 8/10 | **🏆 竞赛营 ¥49,800/¥54,800** |
| 工程方向 | VEX VRC + FRC(入门) + AMC 8 | 智造万物 / 专项营 |
| 编程方向 | USACO白银 + AMC 8 + 科创项目 | 竞赛营（编程类项目） |
| 写作方向 | John Locke Junior + NYT写作赛 | 软实力（大卫老师内容矩阵） |

> **原则**：G6-G8是体制外路线最关键的积累期。竞赛营在这个阶段价值最大——出科研成果、打国际通路、积累申请素材三合一。

### 初中高年级至高一（G8-G9）— 产出期

| 方向 | 重点 |
|------|------|
| 竞赛营出成果 | 青创赛奖项 + 项目材料 + ISEF推荐机会 |
| AMC 10 → AIME | 数学赛道出标志性成绩 |
| USACO白银→黄金 | 编程赛道进阶 |
| FRC组队参赛 | 团队工程大项目的主导经历 |

## 六、回答边界

- 体制外赛事不保证入学结果，只说"有助于差异化申请和学术证明"。
- ISEF 推荐通道需通过青创赛优秀项目选拔，**不能承诺一定能参加 ISEF**。
- 国际赛事信息（赛程、报名方式、费用）以当年官方通知为准。
- AMC、BPhO 等国标赛事国内有授权考局（如 ASDAN），具体信息需查 ASDAN 或对应考点通知。
- 不要用体制外赛事承诺"保奖"或"保录取"。
- 竞赛营的 ISEF 通路价值可以在咨询中说明，但对外传播时注意措辞（不要包装成"竞赛营保证 ISEF 参赛"）。
